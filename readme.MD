# XDC Network RPC Stability check  - code repository


## Step 1 : Clone the repository

```
 git clone https://github.com/GoPlugin/rpc-stability-check.git

```

## Step 2 : do install packages

```
npm install

```

## Step 3 : run the script

```
node index.js
```
